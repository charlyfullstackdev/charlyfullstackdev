## About Me

> Full stack developer operating across the complete software lifecycle — from interface architecture and responsive design to server-side logic, database engineering, and system security. Self-directed in learning, agile in delivery, and intentional in every technical decision.

My development practice is shaped by three core principles:

- **Precision** — code that is purposeful, readable, and maintainable from the first commit
- **Security** — every layer of the stack treated as a potential vulnerability surface
- **Iteration** — continuous improvement applied both to the product and to the developer

> [!NOTE]
> Actively advancing at the intersection of **artificial intelligence** and **cybersecurity** — two domains whose convergence defines the next generation of software engineering.

<details>
<summary><strong>Professional Background</strong></summary>

My trajectory as a developer is defined by structured self-learning, formal methodology training, and a deep commitment to building systems that are not only functional but ***resilient***.

Having developed strong competency in core web technologies — `HTML5`, `CSS3`, and `JavaScript` for the front end, and `Python` for back-end scripting and automation — I expanded into formal agile practice through the *SCRUM* framework, applying it to structure delivery, manage scope, and facilitate iterative refinement cycles.

##### Cybersecurity Integration

Security is not a phase in my workflow — it is a perspective applied throughout every layer of the stack. From input validation and authentication patterns to infrastructure configuration, every architectural decision carries a threat model.
